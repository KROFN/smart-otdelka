# Фоновые изображения для карточек "Виды ремонта"

Положите изображения в эту папку с такими именами:

- `cosmetic.jpg`
- `white-box.jpg`
- `capital.jpg`
- `designer.jpg`

Рекомендации:

- Формат: `jpg`, `jpeg`, `png` или `webp` (если меняете формат, обновите путь в `src/App.tsx`).
- Размер: от `1200x800` и выше.
- Сюжет: спокойные интерьерные фото без слишком светлых участков по центру.

В карточках уже настроен затемняющий градиент поверх фото, чтобы текст всегда оставался читаемым.
